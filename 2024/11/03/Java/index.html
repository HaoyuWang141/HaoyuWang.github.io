Object-Oriented

Class, Abstract Class, Interface

面向对象的三个特性:

  • Encapsulation (封装)
  • Inharitance (继承)
  • Polymorphism (多态)

重要概念:

  • Override (重写/覆写): 父类的实例方法被子类重新实现. 注意: 实例方法为非Static修饰的方法;
  • Overload (重载): 同一个类中, 名字相同但参数列表不同的多个方法;
  • Hide (隐藏): 一个类的 属性\静态方法\内部类 可以分别隐藏其父类中可以访问到的具有相同名字(方法签名)的所有 属性\静态方法\内部类;
  • Shadow (遮蔽): 一个 变量\方法\类 可以分别遮蔽在类内部具有相同名字(方法签名)的 变量\方法\类;
  • 遮掩: 一个变量可以遮掩具有相同名字的一个类, 只要它们都在相同范围内. 相似的, 一个变量或类型可以遮掩一个包.
