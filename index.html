Java keyword transient

Transient

被transient修饰的变量不会被序列化，即不会被保存到文件中。

Volatile

被volatile修饰的变量，保证了线程之间的可见性，即一个线程修改了这个变量的值，其他线程能够立即看到这个修改。

Vim

Vim Commands

Vim Motions

  • w - word forward (W)
  • b - word backward (B)
  • e - end of word (E)
  • ge - end of word backward (gE)
  • f{char} - find character forward
  • F{char} - find character backward
  • t{char} - till character forward
  • T{char} - till character backward
  • ; - repeat last f, t, F, or T command
  • , - repeat last f, t, F, or T command in opposite direction
  • 0 - beginning of line
  • ^ - first non-blank character of line
  • $ - end of line
  • g_ - last non-blank character of line
  • } - next paragraph
  • { - previous paragraph
  • ) - next sentence
  • ( - previous sentence
  • ]] - next section or function
  • [[ - previous section or function
  • ]m - next method
  • [m - previous method
  • ]M - next class or block
  • [M - previous class or block
  • Ctrl+d - down half screen
  • Ctrl+u - up half screen
  • Ctrl+f - down full screen
  • Ctrl+b - up full screen
  • gg - first line
  • {line}gg - go to line {line}
  • G - last line
  • % - matching parenthesis
  • :n - go to line n
  • '' - jump back to the line where the cursor was before the latest jump
  • /{pattern} - search forward for {pattern}
  • ?{pattern} - search backward for {pattern}
  • n - repeat search in same direction
  • N - repeat search in opposite direction
  • * - search forward for the word nearest to the cursor

Vim Editing

  • i - insert before cursor
  • I - insert at the beginning of the line
  • a - insert after cursor
  • A - insert at the end of the line
  • o - insert a new line below the current line
  • O - insert a new line above the current line
  • r - replace one character
  • R - replace multiple characters
  • s - substitute one character
  • S - substitute entire line
