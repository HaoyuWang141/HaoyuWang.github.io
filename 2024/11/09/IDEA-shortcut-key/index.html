IDEA Shortcut Key

新建

Command Description
Alt + Insert 可用于目录树、Java类中
Alt + Enter
Ctrl + Shift + T 生成测试

移动

Command Description
Alt + ↑/↓ 以函数为单位移动
Ctrl + G 定位到行
Ctrl + W 选择单词 => 选择语句 => 选择函数
Ctrl + Shift + W 与上述命令相反
Ctrl + { / Ctrl + } 到大括号的开头和结尾

定位查找

Command Description
Shift + Shift 查找任意文件
Ctrl + N 查找Java类
Ctrl + F 查找
Ctrl + R 替换
F4 进入某一方法或到变量定义处
Ctrl + F12 查看方法和成员变量
Ctrl + Alt + ←/→ 回退/前进到上一次编辑处

编辑

Command Description
Ctrl + D 复制一行
Ctrl + Y 删除一行
Ctrl + / 注释

简单重构

Command Description
Shift + F6 重命名
Ctrl + Alt + M 抽取函数
Ctrl + Alt + V 抽取变量

简单调试

Command Description
Ctrl + Shift + F9/F10 调试/运行测试用例
Ctrl + F8 打断点/取消断点
Shift + F9/F10 调试/运行
F7 进入函数
F8 单步
F9 到下一个断点或技术
Alt + F8 查看表达式结果
